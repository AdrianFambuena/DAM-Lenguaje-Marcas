## 📁 Proyectos Destacados

### 🚀 Archivo Rebelde (Proyecto Star Wars)
Un sitio web temático que incluye:
* **Estructura semántica:** Uso de `header`, `nav`, `main` y `section`.
* **Formularios avanzados:** Sistema de alistamiento con validaciones mediante `pattern` (Regex).
* **Tablas complejas:** Gestión de autorizaciones con uso de `rowspan` y `colspan`.
* **Diseño:** Maquetación limpia utilizando CSS3 (Float, Flexbox).

### 🕹️ Interactividad con JS Nativo
* **Tablero de Ajedrez:** Generación dinámica de elementos mediante bucles en JavaScript.
* **Sistema de Personajes:** Creación de fichas dinámicas con capacidad de inserción y eliminación de nodos.
* **Contador de Victorias:** Manipulación del DOM y gestión de eventos (Click) con cambios de estilo en tiempo real.
